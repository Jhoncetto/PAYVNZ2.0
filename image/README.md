# Carpeta de Imágenes para done for vnz

Aquí puedes cargar las imágenes locales para personalizar la página de ayuda humanitaria del terremoto. El sistema está configurado para leer las imágenes directamente desde esta carpeta.

## Archivos Requeridos:

1. **`hero.jpg`**
   - **Descripción:** Foto principal de la campaña (brigadas de rescate, voluntarios ayudando).
   - **Tamaño Recomendado:** 1200 x 675 píxeles (relación de aspecto 16:9).
   - **Ruta en el sistema:** `/image/hero.jpg`

2. **`medico.jpg`**
   - **Descripción:** Foto de la Dra. Mariana Rojas (Pediatra de Emergencia y Rescate).
   - **Tamaño Recomendado:** 400 x 400 píxeles.
   - **Ruta en el sistema:** `/image/medico.jpg`

3. **`madre.jpg`**
   - **Descripción:** Foto de Carmen Elena Uzcátegui (Madre de Familia Damnificada).
   - **Tamaño Recomendado:** 400 x 400 píxeles.
   - **Ruta en el sistema:** `/image/madre.jpg`

4. **`voluntario.jpg`**
   - **Descripción:** Foto de Francisco Mendoza (Coordinador de Logística de Rescate).
   - **Tamaño Recomendado:** 400 x 400 píxeles.
   - **Ruta en el sistema:** `/image/voluntario.jpg`

## Códigos QR de Pago:
También puedes subir tus códigos QR reales de pago a esta carpeta:
- **`qr_paypal.png`** (configurar en `QR_CONFIG` en `/src/App.tsx` como `/image/qr_paypal.png`)
- **`qr_nequi.png`** (configurar en `QR_CONFIG` en `/src/App.tsx` como `/image/qr_nequi.png`)
- **`qr_breb.png`** (configurar en `QR_CONFIG` en `/src/App.tsx` como `/image/qr_breb.png`)
